logistics 映射

2020-05-07

logistics 映射是一种奇特的映射，结果与初值不敏感，但是随着初值的不同有很多不动点（反例就是洛伦茨吸引子）。具体背景请参见wiki 等。这里只给定义。

xn+1=rxn(1−xn),x1∈(0,1),r∈[0,4]x_{n+1}=rx_n(1-x_n),x_1\in (0,1),r\in [0,4] xn+1​=rxn​(1−xn​),x1​∈(0,1),r∈[0,4]

作出 (r,x∞)(r,x_{\infty})(r,x∞​) 的集合

GeoGebra

序列(序列((r, 迭代(r x (1 - x), x_1, 30)), r, 0, 4, 0.01), x_1, 0.1, 0.9, 0.025)

预览图

 0.01, 1., 0.01}] // Catenate // ListPlot

制作过程：

  1. 建立参数 r,x1,nr,x_1,nr,x1​,n
  2. 计算 r∗x1∗(1−x1)r*x_1*(1-x_1)r∗x1​∗(1−x1​)
  3. 计算 x1+0.1x_1+0.1x1​+0.1
  4. 作点 A(r,x1)A(r,x_1)A(r,x1​)
  5. 迭代 x1→x1+0.1x_1\to x_1+0.1x1​→x1​+0.1 (步长 0.1 )， 次数为 nnn , 只显示终点。
  6. 在终点上取点 BBB，作 BBB 关于 rrr 的轨迹 lll，范围 (0,4)(0,4)(0,4)
  7. 再作 lll 关于 x1x_1x1​ 的轨迹,范围 (0,1)(0,1)(0,1)
